# Usage Key

 			 'a': Add Attraction
             'p': Add Particles
             's': Connect particles (random spring)
             'S': Remove Spring
             'c': Enable collision
             'C': Remove constrain
             'r': show/hide render;
             'f': Add force
             'F': Add great force
             'l': Lock random particles
             'u': Unlock particles
             'x': Enable mouse xy
             'z': Enabvle mouse yz for 3d
             ']': rotate scene right
             '[': rotate scene left
             '+': Add mass to mouse;
             '-': reduce mass to mouse
             'm': mouse collision toggle
